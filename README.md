# Hackathon Utel - Frontend Clean Prototype

Este repositorio contiene la interfaz de usuario (Frontend) para el prototipo desarrollado en la Hackathon Utel, enfocado en un asistente inteligente para construir tiendas en TiendaNube.

Esta versión ha sido simplificada para mostrar únicamente el **Frontend Limpio**, separando CSS y JS de la estructura HTML para una mejor organización y mantenimiento.

## Estructura del Proyecto

- **index.html**: Estructura principal de la aplicación.
- **static/**:
    - **css/style.css**: Estilos profesionales y modernos (Dark Mode).
    - **js/script.js**: Lógica del cliente que simula la interacción con la IA y actualiza el visor en vivo.

## Características

- **Diseño Moderno**: Inspirado en paneles de administración premium.
- **Visor en Tiempo Real**: El código generado (simulado en esta versión estática) se renderiza instantáneamente en un `iframe`.
- **Modo Ver Código**: Permite inspeccionar el HTML/CSS generado por el asistente.

## Cómo empezar

Simplemente abre el archivo `index.html` en cualquier navegador moderno. No se requiere servidor de backend ni dependencias de Python para esta versión.
